## 📋 **Quick Start Guide**

1. **Setup Environment:**
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```

2. **Configure API Key:**
   ```bash
   cp .env.example .env
   # Edit .env with your actual API key
   ```

3. **Run Demo:**
   ```bash
   python demo.py
   ```

4. **Run Tests:**
   ```bash
   pytest tests/ -v
   ```
